## How to Use

1. Clone the repository:

```bash
git clone https://github.com/prerna-kumari83/Cognizant-DotNetFSE-Solutions.git
```

2. Open the required project in **Visual Studio** or **Visual Studio Code**.

3. Restore dependencies:

```bash
dotnet restore
```

4. Build the project:

```bash
dotnet build
```

5. Run the project:

```bash
dotnet run
```

6. For test projects:

```bash
dotnet test
```
